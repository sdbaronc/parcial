Sabías que...

  • Como lenguaje de alto nivel e interpretado, Python no necesita un compilador. Esto es diferente a Java y C ++ que deben compilarse primero antes de ser interpretados. Para Python, se basa en una aplicación conocida como intérprete.
  • A pesar de ser un lenguaje de programación independiente, Python tiene variantes para los lenguajes de programación C y Java. La variante C se conoce como CPython y está diseñada para dar a Python las ventajas de C. Una de estas características es en términos de rendimiento. La variante puede actuar como intérprete y al mismo tiempo como compilador. La variante de Java de Python se conoce como Jython. Aporta algunos aspectos clave de Java, como la productividad, y les permite ejecutarse en una máquina virtual.
  • En el 2012, Oracle y Google se enfrentaron judicialmente, pues Oracle demandaba el uso de APIs del lenguaje Java para realizar la programación de del famoso sistemas operativo móvil "Android".
  • En Javascript Null es un objeto y NaN es un número.
